SSO 핵심 원리

  • • 쿠키 도메인 lvh.me → 모든 서브도메인 공유
  • • 중앙 세션 = 단일 Convex 배포 (직접 만든 인증서버 0줄)
  • • 로그아웃 = 중앙 세션 삭제 → 전 앱 즉시 무효
  • • 세션 기반이라 JWT 무효화 지옥 없음

앱 추가 = 3단계

  • • apps/ 아래 Next.js 앱 복사
  • • /api/auth/[...all] 프록시 (same-origin 필수)
  • • .env.local + 미들웨어 가드
  • • ⚠ authClient는 반드시 자기 프록시 호출 (CORS 회피)
